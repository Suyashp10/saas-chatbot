# SaaS Support Chatbot 🤖

An elementary customer support chatbot built with Python (Flask) for a SaaS product.

## Features
- FAQ matching across 8 categories (35+ FAQs)
- Support ticket creation and tracking
- Human agent escalation
- Intent detection with phrase + token-based matching

## Tech Stack
- Python 3 + Flask
- JSON-based knowledge base and ticket store
- Vanilla HTML/CSS/JS frontend

## Setup
pip install flask
python app.py
